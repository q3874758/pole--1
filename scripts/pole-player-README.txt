PoLE Player Bundle

1. Double-click install-pole-player.cmd.
2. The installer copies pole-client.exe into your user profile.
3. The installer immediately runs player-start:
   - creates the player config if missing
   - captures the current foreground game when possible
   - registers login autostart
   - starts the hidden background daemon

After installation, the player does not need to launch PoLE manually again.
